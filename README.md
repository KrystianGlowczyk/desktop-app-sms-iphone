# desktop-app-sms-iphone
Aplikacja desktopowa na systemy windows służąca do importu oraz prezentacji wiadomości SMS z utworzonego przez program iTunes backupu.
Użytkownik wybiera plik backupu oraz może przenieść wiadomości do lokalnej bazy danych w programie. Aplikacja automatycznie dodaje nowe rekordy po załadowaniu zaaktualizowanego backupu. 
Aplikacja daje możliwość edycji wiadomości oraz kasowania, a także można wyszukiwać wiadomości za pomocą fragmentów wiadomości lub numeru telefonu.

Użyte technologie:
- C#
- .NET Framework
- Window Form
- SQLite
